Hexo Introduction

Hexo là một framework blog tĩnh được viết bằng Node.js.
Với Hexo, bạn có thể viết bài bằng Markdown, sau đó build ra trang web tĩnh cực nhanh, dễ deploy lên GitHub Pages, Vercel, Netlify, Cloudflare Pages hoặc bất kỳ server nào.

Ưu điểm của Hexo

  • ⚡ Nhanh: build hàng trăm bài viết chỉ trong vài giây.
  • 📝 Markdown-first: viết bài dễ dàng bằng Markdown.
  • 🎨 Nhiều theme/plugin: có sẵn kho theme cực lớn (NexT, Butterfly, Icarus, …).
  • 🚀 Triển khai dễ dàng: chỉ cần 1 lệnh hexo deploy là website đã lên mạng.
